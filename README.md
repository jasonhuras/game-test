# Game Design Repository

## Project Codename: NIGHTFALL (Placeholder)

### Overview
A 3D isometric action roguelike combining V Rising's combat depth, Vampire Survivors' addictive gameplay loop, and TFT's strategic augment system.

### Core Pillars
1. **Satisfying Combat** - V Rising-inspired weapons and spell combos
2. **Addictive Loop** - Wave-based survival with meta progression
3. **Strategic Depth** - TFT-style augment choices that define builds
4. **Competitive Edge** - Ranked 1v1/2v2 PvP survival mode

### Repository Structure
